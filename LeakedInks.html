MANGO DB

The Best Mem Lib Database.

Welcome, Mango DB is a memory leak database.Is a time saver & a powerfull tool.
What you can do with it is unlimited.It work in as a online database no download
needed, no sign up etc...How you can use it? Well Mango DB as been thinked for
programmers and work with list and map linker.Down below theres a how to link
your script to the database.

To link your script to the database use "connect_service".
and connect with the "user:mangodb" "pass:public" on the port
"22".

once your script is connected to the database. Use "host_computer"
with ".File(value)", the value can be

  • /server/mangodb/memlib/http
  • /server/mangodb/memlib/ssh
  • /server/mangodb/memlib/sql


Here an example of use.

mango_db = get_shell.connect_service("56.35.45.131",22,"mangodb","public","ssh")
http_mem = mango_db.host_computer.File("/server/mangodb/memlib/http").get_files
for x in http_mem
print(x.name)
end for
exit
